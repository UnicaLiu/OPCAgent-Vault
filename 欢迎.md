# 👋 欢迎来到 OPCAgent 量化知识库

这是 **OPCAgent** 的知识库——一个 AI 驱动的电商量化跑品系统。

## 🚀 快速开始

1. **了解知识库结构** → [[wiki/index|wiki/index.md]]
2. **阅读行为规则** → [[CLAUDE]]
3. **查看子代理定义** → [[AGENTS]]

## 📥 如何投放资料

将原始资料放入 `raw/` 对应目录：
- 产品图片/规格 → `raw/产品原始资料/`
- 竞品信息 → `raw/竞品参考/`
- 爆款视频素材 → `raw/爆品素材/`
- 外部参考文章 → `raw/外部知识/`

Agent 会自动检测并编译到 `wiki/` 中。
