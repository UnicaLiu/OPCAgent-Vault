# 爆品分析

> 分析 TikTok/电商平台爆款视频，提炼可复用的内容策略、视频结构和素材元素。
> 此板块是产品报告的"认知输入源"——给销售经理增加认知脑容量。

---

## 分析报告 `分析报告/`

每个分析报告包含：

| 维度 | 说明 |
|------|------|
| 视频基本信息 | 平台、时长、发布时间、互动数据 |
| 结构拆解 | 开头钩子、信息递进、CTA 设计 |
| 文案分析 | 文案公式、关键词、情绪节奏 |
| 视觉分析 | 画面风格、转场手法、色彩调性 |
| 音频策略 | BGM 选择、音效使用、语速节奏 |
| 可复用元素 | 值得借鉴的打法和元素 |
| 适配建议 | 如何应用到我们的产品 |

## 素材索引 `素材索引/`

特征标注索引：
- **文案公式索引**：按公式类型分类（AIDA / PAS / 痛点-解决 / ...）
- **视觉风格索引**：按视觉风格分类（UGC / 专业棚拍 / 生活场景 / ...）
- **品类打法索引**：按品类交叉索引有效打法
