# OPCAgent 量化知识库

> **OPCAgent** — AI 驱动的电商量化跑品系统知识库。
> 以「产品」为唯一业务源头，串联：产品信息 → AI 报告 → 任务编排 → 脚本生成 → 视频制作 → 数据归因。

## 🔧 快速操作

| 操作 | Agent 子代理 | 说明 |
|------|-------------|------|
| 摄入新产品 | `product-ingestor` | 读取 raw/ 产品资料 → 创建产品档案 |
| 生成产品报告 | `report-generator` | 读取产品档案 → 生成版本化报告 |
| 分析爆款视频 | `hot-product-analyzer` | 读取爆品素材 → 生成分析报告 |
| 记录任务 | `task-recorder` | OPCAgent 创建任务 → 记录 + 追踪 |
| 归档脚本 | `script-archiver` | 脚本生成完成 → 归档 + 关联 |
| 归因分析 | `attribution-analyzer` | 投放数据 → 归因报告 + 洞察 |
| 健康检查 | `wiki-linter` | 断裂链接 / 孤儿页面 / 索引同步 |
| 知识同步 | `knowledge-syncer` | 多主机公共专家库同步 |
