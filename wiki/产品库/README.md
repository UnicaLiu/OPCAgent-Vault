# 产品库

> OPCAgent 管理的所有产品档案和 AI 生成的版本化产品报告。
> **产品是唯一业务源头**，一切数据可追溯到产品。

---

## 产品档案 `产品档案/`

每个产品一个 .md 文件，包含完整的产品认知：

| 字段 | 说明 |
|------|------|
| 产品名称 | 中文名 + 本地语言名 |
| 目标国家 | 投放的目标市场列表 |
| 目标语言 | 脚本/字幕语言 |
| 分类 | 三级品类路径 |
| 价格 / SKU | 售价和规格 |
| 核心卖点 | 关键 selling points |
| 禁忌偏好 | 文化禁忌、受众偏好 |
| 产品图片 | 引用 raw/ 中的图片 |
| 报告索引 | 关联的产品报告版本列表 |

## 产品报告 `产品报告/`

版本化的 AI 分析报告：

```
{产品名}-产品报告-v1.md
{产品名}-产品报告-v2.md
...
```

每次生成新版本时：
- 新报告 `is_active: true`
- 旧版本 `is_active: false`（保留历史）
- 更新产品档案的 `report_versions` 字段
