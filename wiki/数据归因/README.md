# 数据归因

> 投放数据归因分析和数据洞察提炼。
> 此板块是产品报告的"认知输入源"——给销售经理更深层、更正确的果实。

---

## 归因报告 `归因报告/`

每个归因报告包含：

| 维度 | 说明 |
|------|------|
| 投放概览 | 投放周期、预算、平台、产品 |
| 核心指标 | CTR / CVR / ROAS / GMV / CPA |
| 受众分析 | 转化人群画像、兴趣标签分布 |
| 素材效果 | 各素材版本的对比表现 |
| 时段分析 | 最佳投放时段和日期 |
| 归因路径 | 用户从曝光到转化的关键路径 |
| 问题诊断 | 低效环节识别和原因分析 |

## 数据洞察 `数据洞察/`

从归因数据中提炼的可操作认知：

| 类型 | 说明 |
|------|------|
| 产品认知验证 | 哪些卖点/定位被市场验证有效 |
| 受众洞察 | 实际购买者的特征 vs 假设的目标受众 |
| 素材优化方向 | 基于数据的高效素材改进建议 |
| 投放策略调整 | 出价/定向/预算的优化建议 |
| 反哺产品报告 | 应更新产品报告的认知结论 |
