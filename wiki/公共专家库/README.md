# 公共专家库

> 此板块内容在**所有部署主机上保持一致**，由核心团队维护，定期同步。

## 目录

### [[平台规则/]]

- TikTok 广告政策
- Shopee 平台规则
- Amazon 广告规范
- 各平台算法更新日志

### [[投放策略/]]

- 冷启动策略
- 扩量方法论
- 受众定向技巧
- 素材疲劳度管理

### [[内容方法论/]]
视频内容的创作公式、脚本结构、文案框架。

- AIDA 公式
- PAS 公式
- 黄金前 3 秒
- 分镜设计原则

### [[工具指南/]]
OPCAgent 及周边工具的使用手册。

- OPCAgent 系统操作指南
- AI CLI 配置说明（Claude Code / Codex）
- FastAPI 后端部署
- 环境探针使用
